# NeoLogicIA Facebook Config App

Este repositorio contiene páginas públicas necesarias para la configuración y aprobación de la aplicación en Meta (Facebook Developers).

## 🌐 Descripción

Esta aplicación permite automatizar publicaciones en Facebook utilizando inteligencia artificial.

## 📄 Contenido

- index.html → Página principal
- privacy.html → Política de privacidad
- terms.html → Términos de servicio

## 🔒 Privacidad

No recopilamos datos personales sin consentimiento.  
La información se usa únicamente para automatizar publicaciones en Facebook.
